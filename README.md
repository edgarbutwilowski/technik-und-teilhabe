# Blog: Technik & Teilhabe

Wissenschaft und Technik sind Teil unseres gesellschaftlichen Fundaments. Dieses Repository enthält den Quellcode und die Markdown-Dateien für den Quarto-Blog **[Technik & Teilhabe](https://edgarbutwilowski.github.io/technik-und-teilhabe/)**.

👉 **[Hier geht es direkt zum veröffentlichten Blog](https://edgarbutwilowski.github.io/technik-und-teilhabe/)**

Es geht bei diesen Projekten weniger um fertige Antworten als um den Prozess des Verstehens – vom Umgang mit Geodaten über das Bauen eigener Sensoren bis hin zur Frage, wie wir digitale Resilienz in einer komplexen Welt bewahren.

### Projektbereiche

* **Geoinformation & Raum:** Analysen und Konzepte zur digitalen Stadt.
* **Sensorik & IoT:** Dokumentationen zu Feinstaub-Messungen, Urban Farming und Telemetrie.
* **Digitale Resilienz:** Notizen zu Souveränität und technischer Selbstverteidigung.

### Für Mitwirkende und die technische Dokumentation

Wenn Sie den Blog lokal ausführen möchten, benötigen Sie eine Installation von [Quarto](https://quarto.org/). 
Klönen Sie das Repository und führen Sie im Hauptverzeichnis folgenden Befehl aus, um die lokale Vorschau zu starten:
```bash
quarto preview
```
Die Live-Veröffentlichung geschieht automatisch über GitHub Actions, sobald Änderungen in den `main`-Branch gepusht werden.
